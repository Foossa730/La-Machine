# Sons personnalisés

Place ici tes extraits audio autorisés, en gardant ces noms de fichiers :

- `intro.mp3` : son au lancement du quiz
- `correct.mp3` : son pour une bonne réponse
- `wrong.mp3` : son pour une mauvaise réponse
- `unlock.mp3` : son quand un code ou une récompense est débloqué

Si un fichier manque, le quiz utilise automatiquement le son synthétique déjà intégré.
